# MulVAL-MITRE
"MulVAL Interaction Rules.xlsx" includes all the MulVAL interaction rules

"ATT&CK Enterprise Techniques with MulVAL IR.xlsx" includes the mapping of MulVAL IRs to MITRE ATT&CK Techniques

SIRGen directory includes a helper app for generating new MulVAL set of interaction rules (SIR) for ATT&CK Technique scenario
